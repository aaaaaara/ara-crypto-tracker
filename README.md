## Front-end Boiler Plate(React/Web)

### Tech Stack
- react v17
- react-router-dom (v6)
- typescript (v4.5)
- React Query (v3)
- Recoil (v0.6)
- Styled-Components (v5)
- Framer-Motion (v6)
- StoryBook 

### Build 
- Web: npm start
- Storybook: npm run storybook

### 환경변수
- 개인 KEY값, DEV URI, MAIN URI ...
- .env 파일로 관리
- gitignore 파일에 추가
