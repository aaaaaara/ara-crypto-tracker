## Front-end Boiler Plate(React/Web)

### Tech Stack
- react
- react-router-dom
- typescript
- React Query
- Recoil
- Styled-Components
- StoryBook
- Framer-Motion

### Build 
- Web: npm start
- Storybook: npm run storybook

### 환경변수
- 개인 KEY값, DEV URI, MAIN URI ...
- .env 파일로 관리
- gitignore 파일에 추가
